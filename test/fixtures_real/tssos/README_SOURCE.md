# TSSOS Raw Fixture Snapshot

This directory contains committed raw sparse SOS frontend snapshots for CertSDP 3.0 importer replay. The raw JSON is intentionally not CertSDP certificate schema: it preserves TSSOS-style variables, objective, constraints, relaxation order, correlative sparsity cliques, monomial bases, Gram/localizing blocks, rationalized entries, and untrusted solver metadata.

The snapshots are generated by `generate_raw_tssos_fixture.jl` from the exact offline fixture data already committed in this repository. In a live TSSOS.jl session the same shape corresponds to exported structured moment-SOS data after rationalization. Solver status, residuals, and frontend metadata are ignored by CertSDP; the normalized certificate must pass Gram/basis/localizing exact replay and DAG replay.

Regeneration does not require commercial solvers, internet, or TSSOS in CI. The source hash is recomputed by the importer from raw JSON.
