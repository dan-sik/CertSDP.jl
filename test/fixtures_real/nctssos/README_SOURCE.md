# NCTSSOS / Quantum Raw Fixture Snapshot

This directory contains committed raw noncommutative/trace frontend snapshots for CertSDP 3.0 importer replay. The raw JSON is not CertSDP certificate schema: it preserves NC variables, word bases, quotient relations, moment blocks, objective data, rewrite witnesses, and untrusted solver metadata.

The snapshots are generated by `generate_raw_nctssos_fixture.jl` from deterministic offline fixture data. In a live NCTSSOS-style export the same shape represents trace/NPA moment data after rationalization. CertSDP imports it as untrusted candidate data and accepts only after rewrite-witness replay, moment PSD replay, objective reconstruction, and DAG replay.
